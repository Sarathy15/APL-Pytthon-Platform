# APL to Python Migration Platform

Enterprise-grade automation for legacy logic refactoring.

## Setup
1. **Backend**: `pip install -r requirements.txt`
2. **Frontend**: `npm install`
3. **Model**: `ollama pull qwen2.5-coder:7b`

## Architecture
- **Agents**: Asynchronous AI collaborative pipeline.
- **Validation**: High-precision dual-execution checks.
- **Reporting**: PDF/JSON export for compliance.
