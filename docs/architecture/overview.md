# Enterprise Migration Architecture

## System Components
1. **Ingestion Layer**: Handles APL file uploads and metadata.
2. **Agentic Core**: Multi-agent pipeline (Understanding, Conversion, Validation).
3. **Execution Sandbox**: Dual-runtime for behavioral comparison.
4. **Scoring Engine**: Multi-vector reliability assessment.

## Security
- Local-only Inference (Ollama/DeepSeek).
- Encrypted trace storage.
- Role-based access control.
